BizOps Configurator Overview

This is a Single Page App (SPA) to get inputs from the user and from the Dynatrace API, to transform a stock collection of dashboards, and to upload via the Dynatrace API

BizOps Dashboard IDs

For BizOps Dashboards we predefine the dashboard ID to make it easier to track the dashboards as a collection.
Example: bbbbbbbb-0001-0001-0001-0000000000001
  • bbbbbbbb - Prefix to denote BizOps Dashboards
  • 0001 - Tenant Overview
  • 0001 - Application Overview
  • 0001 - Funnel Overview
  • 0000000000001 - Dashboard number

Reserved Dashboard IDs

  • bbbbbbbb-XXXX-0000-0000-000000000000 - Tenant Overview Dashboard
  • bbbbbbbb-XXXX-XXXX-0000-000000000000 - App Overview Dashboard
  • bbbbbbbb-XXXX-0000-0000-ffffffffffff - Tenant Overview Config
  • bbbbbbbb-XXXX-XXXX-0000-ffffffffffff - App Overview Config
  • bbbbbbbb-XXXX-XXXX-XXXX-ffffffffffff - Funnel Config
