      5. BizOps Dashboard IDs

      For BizOps Dashboards we predefine the dashboard ID to make it easier to track the dashboards as a collection.
      Example: bbbbbbbb-0001-0001-0001-0000000000001
      • bbbbbbbb - Prefix to denote BizOps Dashboards
      • 0001 - Collection number, increments with each deployment
      • 0001 - BizOps Dashboards version
      • 0001 - Application number, for multiple apps in a collection
      • 0000000000001 - Dashboard number, increments for each dashboard in the collection
